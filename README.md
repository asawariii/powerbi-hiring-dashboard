# powerbi-hiring-dashboard

This Power BI dashboard provides a comprehensive view of hiring performance across regions, store sizes, and recruitment channels. It is designed to help HR teams and business leaders analyze hiring patterns, optimize recruitment strategies, and monitor key workforce metrics in real time.

Key Features:

Total Hires Overview: Displays the total number of hires across the selected period.
Hiring Cost by Region: Visual breakdown of recruitment expenses by region.
Hiring Trends Over Time: Line chart showing year-over-year hiring activity.
Hiring Mode Analysis: Identifies the most effective recruitment channels (employee referrals, job portals, social networks, etc.).
Store Size Hiring Insights: Distribution of hires across small, medium, and large store categories.
Interview-to-Hire Funnel: Visual representation of interview stages and conversion rates.
Geographical Hiring Map: Highlights hiring density across locations using an interactive map.
Purpose:

This template helps organizations:
Track hiring performance
Identify cost-effective recruitment sources
Understand hiring distribution across locations
Support data-driven talent acquisition decisions
Tools Used: Power BI, DAX, Data Modeling, Visualization Design
